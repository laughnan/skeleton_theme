<h1>Skeleton Boilerplate</h1>
<p>Adapted from Skeleton (http://getskeleton.com/), this project is meant to be a boilerplate, SASS-centric Drupal theme providing basic styling and a grid system.</p>

<h3>NOTES</h3>
<p>
<ul>
<li>The <code>css</code> and <code>.sass-cache</code> directories are hidden by default. To generate the CSS, use compass.</li>
</ul>
</p>
